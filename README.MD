#JAVA通用maven-java代码

发布应用程序的时候注意如下几点：（后续会增加很多功能）
1. framework-类的是基础包，是必须要加上
2. plugin- 类的是插件包，需要那些插件就加上那些插件，但是必须将各自工程下的resources的配置文件放入framework-web下
3. project- 类是具体的项目，可以选择性加入，对应各自项目，也必须将各自工程下的配置文件放入web中去
